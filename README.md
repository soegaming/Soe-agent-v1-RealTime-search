# Soe Agent v1 - Real-Time Search

AI Agent CLI yang bisa search Google real-time pake Gemini API.

## Features
- Real-time web search via Gemini 1.5 Flash
- Command line interface 
- Secure API key pake.env

## Setup
1. git clone https://github.com/soegaming/Soe-Agent-v1-RealTime-search.git
2. pip install google-generativeai python-dotenv
3. Bikin.env: GEMINI_API_KEY=your_api_key
4. python 01-cli-agent.py
